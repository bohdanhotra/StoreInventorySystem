## 🎯 Про проєкт
Це сучасний настільний додаток, розроблений для зручного управління каталогом. Він дозволяє вести облік залишків, швидко знаходити потрібні позиції та зберігати дані локально (JSON) без необхідності піднімати окрему базу даних.

## ✨ Ключові можливості
* 🔐 **Безпека:** Авторизація та реєстрація з безпечним хешуванням паролів.
* 🛠 **Управління (CRUD):** Додавання, редагування, перегляд та видалення товарів.
* 🔍 **Пошук та фільтри:** Швидка навігація за назвою та категоріями.
* 📄 **Експорт:** Вивантаження поточного стану складу у форматі `CSV`.
* 🎨 **Персоналізація:** Підтримка **світлої ☀️** та **темної 🌙** тем оформлення.
* 🌍 **Локалізація:** Інтерфейс адаптовано 🇺🇦 українською та 🇬🇧 англійською мовами.

## 🚀 Швидкий старт
1. Клонуйте репозиторій на свій ПК.
2. Відкрийте файл `StoreInventorySystem.slnx` у **Visual Studio 2022**.
3. Запустіть проєкт (`F5`).

> 💡 *Базові файли (users.json, products.json, settings.json) створяться автоматично у папці з програмою під час першого запуску.*
